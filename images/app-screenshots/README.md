# Capturas de Pantalla de la App Móvil

Esta carpeta contiene las capturas de pantalla de la aplicación móvil que se muestran en la landing page.

## Imágenes Requeridas

Por favor, añade las siguientes imágenes con estos nombres exactos:

1. **inicio.png** - Pantalla de inicio con lista de huéspedes ("Quién hay hoy")
2. **nueva-reserva.png** - Modal de nueva reserva
3. **reservas.png** - Pantalla de lista de reservas
4. **calendario.png** - Pantalla de calendario con reservas
5. **enlaces-pago.png** - Modal de nuevo enlace de pago

## Especificaciones

- **Formato**: PNG (recomendado) o JPG
- **Dimensiones recomendadas**: 300x650px (proporción iPhone)
- **Tamaño máximo**: 500KB por imagen
- **Calidad**: Alta resolución para que se vea bien en el frame de iPhone

## Nota

Si las imágenes no están disponibles, la landing page mostrará placeholders automáticamente.
